# 2026-05-13 Hausübung

Ein simples Console-Projekt mit EF Core und SQLite im Strava-Kontext.

## Enthaltene Tabellen

- Athleten
- Aktivitäten

## Funktionen

- Athleten anlegen, anzeigen, bearbeiten, löschen
- Aktivitäten anlegen, anzeigen, bearbeiten, löschen
- Beziehung: Ein Athlet hat mehrere Aktivitäten

## Starten

```bash
dotnet run --project 2026-05-13_Hausuebung.csproj
```
